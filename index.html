penjelasan

  • shift(): Metode shift() digunakan untuk menghapus elemen pertama dari sebuah array dan menggeser elemen-elemen lainnya ke indeks yang lebih rendah. Metode ini mengembalikan elemen yang dihapus.

  • unshift(): Metode unshift() digunakan untuk menambahkan satu atau lebih elemen ke awal array dan menggeser elemen-elemen lainnya ke indeks yang lebih tinggi. Metode ini mengembalikan panjang baru dari array tersebut.

  • pop(): Metode pop() digunakan untuk menghapus elemen terakhir dari sebuah array dan mengembalikan elemen tersebut. Metode ini mengubah panjang array.

  • push(): Metode push() digunakan untuk menambahkan satu atau lebih elemen ke akhir array dan mengembalikan panjang baru dari array tersebut.
